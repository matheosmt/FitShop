# FitShop - E-commerce Web Application

[![FitShop Logo](path_to_logo_or_screenshot.png)](https://matheosmt.github.io/FitShop/index.php)

## Project Overview
FitShop is a modern e-commerce web application for selling sports equipment, supplements, and accessories. This project was developed as part of the **BTS SIO** curriculum to demonstrate practical web development skills.

The application includes:
- User authentication (login and account management)
- Product catalog with categories
- Shopping cart functionality
- Customer reviews section
- Contact form

This project was built using **PHP, HTML, CSS, and JavaScript**.

---

## Features
- **User Accounts:** Create an account, log in, and manage your profile.  
- **Product Catalog:** Browse products by category, view details, and add to cart.  
- **Shopping Cart:** Add or remove items, view total prices.  
- **Customer Reviews:** Leave and read reviews for products.  
- **Contact Form:** Send messages directly from the website.  

---

## Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript  
- **Backend:** PHP  
- **Session Management:** PHP Sessions  
- **Data Storage:** Currently using PHP arrays (can be extended to MySQL)
